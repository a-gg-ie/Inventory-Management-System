This project is a console-based Inventory Management System developed using Python, MySQL, and CSV file handling. It is designed to help manage product details, suppliers, orders, and sales records efficiently through a menu-driven interface.
The system allows users to perform essential inventory operations such as adding products, viewing and searching inventory, managing suppliers, placing orders, and recording sales transactions. Data is stored securely in a MySQL database, while product details are also maintained in a CSV file for easy access and backup.
